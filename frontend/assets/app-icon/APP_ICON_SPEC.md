# Ora AI App Icon Specification

## Design Overview
Simple, recognizable icon using Ora logomark on solid background. Follows Apple Human Interface Guidelines.

## Light Mode Variant
- **Background**: Sage green (#8B9A7E) - from Ora brand palette
- **Logo**: White Ora logomark (circular symbol)
- **Style**: Soft, rounded, calming aesthetic
- **Size**: 1024x1024px master

## Dark Mode Variant
- **Background**: Deep navy (#1A2332)
- **Logo**: Cream/off-white (#F5F3ED) Ora logomark
- **Style**: Same as light mode, optimized for dark UI
- **Size**: 1024x1024px master

## Required Export Sizes (iOS)
- 1024x1024 (App Store)
- 180x180 (iPhone)
- 167x167 (iPad Pro)
- 152x152 (iPad)
- 120x120 (iPhone @2x)
- 87x87 (iPhone @3x notification)
- 80x80 (iPad @2x)
- 76x76 (iPad)
- 58x58 (iPhone @2x spotlight)
- 40x40 (iPad)
- 29x29 (Settings)

## Design Principles
1. **Simple**: Single logomark, no text
2. **Recognizable**: Works at small sizes (29px)
3. **Distinct**: Unique shape stands out in App Store
4. **On-brand**: Uses official Ora colors and logomark
5. **Rounded corners**: iOS automatically applies mask
