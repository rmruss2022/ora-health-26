## Audio Specifications
- **Sample Rate**: 44.1 kHz
- **Bit Rate**: 128-192 kbps (good quality, reasonable size)
- **Channels**: Stereo
- **Format**: MP3 or AAC
- **Looping**: Must loop seamlessly (fade in/out at edges if needed)
