## Overview

## Design System

### Typography Overlay
- **Headline Font:** Sentient Bold, 40-48pt
- **Subtext Font:** Switzer Regular, 22-26pt
- **Color:** White text with subtle shadow for readability
- **Alignment:** Left-aligned or centered depending on composition

### Device Frame
- Use realistic iPhone frame (silver or graphite)
- Subtle shadow for depth
- No distracting backgrounds

### Background Gradients
Each screenshot has a unique gradient background that:
- Complements the screen content
- Uses brand colors (sage, olive, terracotta, cream, mustard)
- Provides visual variety across the set
- Creates cohesion as a series

### Real Content
- No "lorem ipsum" or placeholder text
- Use actual app screens captured on device
- Ensure all text is legible and professional
- Remove any test data or debug information

## Creation Process

### Step 1: Capture Screens
1. Build app in Release mode
2. Run on iPhone 15 Pro Max simulator
3. Navigate to each key screen
4. Use Simulator ‚Üí File ‚Üí Save Screen (‚åòS)
5. Capture at actual device resolution

### Step 2: Edit in Design Tool
1. Open Figma/Photoshop
2. Create artboard: 1290 x 2796px
3. Place screenshot
4. Add device frame
5. Add gradient background (behind frame)
6. Add text overlays with proper typography
7. Ensure text is readable (test at thumbnail size)

### Step 3: Export
- Format: PNG (RGB, 8-bit)
- Resolution: Actual pixels (1290 x 2796)
- Color Space: sRGB
- No compression
- File size: <10MB each

### Step 4: Preview
- View at thumbnail size (160px wide)
- Ensure key message is visible
- Check readability on both light and dark App Store backgrounds
- Test on actual device if possible
