# Ora Health - OpenClaw Project

## Overview
Mobile health & wellness app with AI assistant

## Structure
- `ora-health/` - React Native mobile app
- `ora-health-api/` - Node.js backend API

## Quick Start
```bash
cd ora-health && npm start
cd ora-health-api && npm run dev
```
