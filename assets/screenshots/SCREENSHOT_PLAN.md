## Design Guidelines
- **Device frame**: Use official Apple device frames (white iPhone)
- **Background gradient**: Soft, on-brand gradient behind device
- **Typography**: Sentient for headlines, Switzer for body text
- **Headline placement**: Above device (centered)
- **Subtext**: Below headline (smaller, gray)
- **Padding**: 80px top/bottom margins, centered device
- **Export format**: PNG, 72dpi, sRGB color space

## Implementation Notes
Screenshots should be captured from actual app running on simulator at appropriate resolution.
Add device frame and marketing text in design tool (Figma/Photoshop).
